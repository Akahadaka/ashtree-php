ashtree-php
===========

Homebrew PHP Framework

Usage
=====

vim Vagrantfile
change name to your project name
:wq

vagrant up
navigate to http://192.168.56.101/ to see the page
or modify your hostfile

sudo vim /etc/hosts
add line 192.168.56.101  alpha.dev
:wq
navigate to http://alpha.dev/ to see page
